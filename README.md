# Intelligent Vision-Based Examination Monitoring System (IVB-EMS)

## Project Overview
This project is an AI-driven security solution designed to automate proctoring for online examinations, ensuring academic integrity through real-time monitoring.

## Features
* **Object Detection:** Powered by **YOLOv8** to identify unauthorized objects like mobile phones and books.
* **Behavioral Analysis:** Uses **MediaPipe** to track head-pose estimation and gaze patterns.
* **Automated Logging:** Captures time-stamped visual evidence of infractions for administrative audit.

## Tech Stack
* **Language:** Python
* **Libraries:** OpenCV, YOLOv8, MediaPipe
* **Data Logging:** SQL/Excel integration
